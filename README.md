# ELD Variedades

Site da loja **Eunice Luzia Dantas — Variedades**, desenvolvido com React, TanStack Router e Supabase.

🔗 **Deploy:** [eldvariedades.lovable.app](https://eldvariedades.lovable.app/)

---

## Sobre

Catálogo online de roupas, perfumes e variedades. O cliente navega pelos produtos, monta o carrinho e finaliza o pedido diretamente pelo WhatsApp — simples, rápido e pessoal.

## Tecnologias

- **React** + **TypeScript**
- **TanStack Router** — roteamento
- **TanStack Query** — gerenciamento de dados
- **Supabase** — banco de dados e autenticação
- **Tailwind CSS** — estilização
- **Vite** — build

## Funcionalidades

- Catálogo de produtos com categorias e filtros
- Página de produto com seleção de tamanho e cor
- Carrinho com finalização via WhatsApp
- Painel administrativo (acesso restrito)
- Layout responsivo para mobile e desktop

## Rodando localmente

```bash
npm install
npm run dev
```

## Painel Admin

Acesso restrito. Login em `/login`.
