# ELD Variedades 🛍️

Loja virtual de variedades com catálogo de produtos, carrinho de compras e painel administrativo.

🔗 **Acesse o site:** [https://eldvariedades.joaodouglasdantas02b.workers.dev/](https://eldvariedades.joaodouglasdantas02b.workers.dev/)

---

## Tecnologias

- **[TanStack Start](https://tanstack.com/start)** — framework React fullstack com roteamento baseado em arquivos
- **[React 19](https://react.dev/)** — biblioteca de interface
- **[Supabase](https://supabase.com/)** — banco de dados PostgreSQL, autenticação e storage
- **[Tailwind CSS v4](https://tailwindcss.com/)** + **[shadcn/ui](https://ui.shadcn.com/)** — estilização e componentes
- **[Cloudflare Workers](https://workers.cloudflare.com/)** — deploy e hospedagem na edge
- **[Vite](https://vite.dev/)** — bundler e servidor de desenvolvimento

---

## Funcionalidades

- Catálogo de produtos com imagens, categorias e filtros por gênero
- Página de detalhe do produto com galeria de imagens
- Carrinho de compras
- Autenticação de administrador
- Painel admin para gerenciar produtos e categorias

---

## Rotas

| Rota | Descrição |
|------|-----------|
| `/` | Página inicial com produtos em destaque |
| `/produtos` | Catálogo completo |
| `/produto/:id` | Detalhe do produto |
| `/carrinho` | Carrinho de compras |
| `/login` | Login do administrador |
| `/admin` | Painel administrativo |
| `/admin/produtos` | Gerenciamento de produtos |
| `/admin/categorias` | Gerenciamento de categorias |
